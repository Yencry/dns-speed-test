# dnsSpeedTest

纯前端 DNS 延迟测试工具。在本地浏览器中直接对各个 DNS over HTTPS (DoH)端点测速，帮助你选择更快、更稳定的解析服务。

---

## 包含了哪些 DoH 端点？

### 基础列表

- **大型企业 / 公共 DNS**
  - Cloudflare
  - Google
  - AdGuard
  - ControlD
- **政府 / 教育 / 非营利**
  - Canadian Shield
  - DNS0.eu
  - Quad9
- **隐私与技术社区项目**
  - Mullvad
  - Digitale Gesellschaft
  - Applied Privacy
  - CleanBrowsing
  - 其它社区维护端点（BlahDNS、LibreDNS、DNSWarden 等）
